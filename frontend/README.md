# Movie Recommender Frontend

Vue 3 + Vite single-page app for the recommender backend.

## Prerequisites
- Node.js 18+

## Install
The environment may block direct npm access. If the default registry fails, set a mirror:
```bash
npm install --registry=https://registry.npmmirror.com
```

## Run
```bash
npm run dev
```

Set `VITE_API_BASE` in a `.env` file if the backend is not running on `http://localhost:8080/api`.
